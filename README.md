# SPYLT Website Clone

A recreation of the SPYLT caffeinated chocolate milk website, featuring an energetic and engaging design that showcases the brand's bold personality and product lineup.

## About SPYLT

SPYLT is a brand of high-protein, sugar-free, caffeinated chocolate milk designed to provide a delicious pick-me-up with 60mg of caffeine and 20g of protein per serving. The brand embraces a rebellious, fun-loving attitude with the tagline "Chill it, then chug it!"

## Project Overview

This project is a clone implementation of the SPYLT website (spylt.com), recreating its vibrant, interactive design and smooth animations. The website features a modern, colorful aesthetic with dynamic scrolling effects and engaging user interactions.

## Features

- Responsive design optimized for all devices
- Smooth scroll animations and transitions
- Interactive product showcase sections
- Hero section with compelling brand messaging
- Flavor selection display
- Nutrition and benefits information
- Testimonials section
- Footer with newsletter signup and navigation
- Video integration for product promotion

## Tech Stack

- **React.js** - Component-based UI development
- **CSS** - Custom styling and animations
- **JavaScript** - Interactive functionality

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd spylt-clone
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```
